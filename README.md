📄 Project Title

Agentic Multi-Agent AI System for Autonomous Cloud Security

Description

This project simulates an agentic AI system for cybersecurity in cloud authentication environments. It uses a multi-agent architecture to perform:

Anomaly detection
Risk assessment
Decision-making (ALLOW / MFA / BLOCK)
Automated response actions

System Architecture

Detection Agent → identifies anomalies
Decision Agent → evaluates risk
Response Agent → executes actions
Logger Agent → records results

Technologies Used

Python
Google Colab
Rule-based AI simulation
Multi-agent system design
